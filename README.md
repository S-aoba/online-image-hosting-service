# online-image-hosting-service
## 概要
ユーザー登録不要のオンライン上で画像を共有し合えるサービスです。

## 説明
本サービスの特徴は以下となります。
- **シームレスな利用体験**: ユーザー登録の手間が不要で、わかりやすいUI画面を提供しています。直感的な操作で、誰でも簡単に利用できます。

- **手軽な共有**: 画像をアップロードして発行される共有用URLを使えば、友達や家族と簡単に画像を共有できます。さまざまなシーンでの共有が可能です。

- **セキュリティ対策**: 悪用を防ぐために、アップロードレートをIPアドレスごとに制限するなどの対策を講じています。安心して利用できる環境を提供しています。

- **スケーラビリティ**: 画像はスケーラビリティを意識した構築をしており、さまざまなデバイスや解像度での閲覧に対応しています。

- **多様な拡張子に対応**: png、jpeg、jpg、gifの拡張子に対応しており、さまざまな形式の画像をアップロードできます。

- **容量制限**: 各ユーザーごとに一日最大5MB、一回の最大容量3MBまでの画像保存が可能です。十分な容量を確保しつつ、サービスの効率的な運用を実現しています。

- **閲覧回数**: ビューカウンターを備えており、画像が閲覧された回数をカウントします。
